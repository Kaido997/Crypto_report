Requisiti:

1. Modulo requests
2. Una API KEY da https://pro.coinmarketcap.com/
3. Python 3

Funzionalita:

Questo script crea un file json che a come nome la data e ora del momento in cui si lancia contenente delle informazioni sul mercato delle criptovalute

Le informazioni:

1. La top 20 delle criptovalute per capitalizzazione 
2. La criptovaluta con il volume maggiore
3. Le migliori e peggiori 10 me percentuale in 24 ore
4. Il denaro necessario per acquistare una unita di ogni singola criptovaluta nella top20
5. La percentuale di guadagno se acquistato una unita della top20 il giorno prima
